## What is VisionCamera?

VisionCamera is a React Native library that exposes the full native camera APIs on iOS (AVFoundation) and Android (Camera2) through a clean, declarative interface. It is built on a custom C++/GPU-accelerated video pipeline and supports everything from simple photo capture to realtime frame processing with JavaScript worklets.

## Minimal example

```tsx
import { Camera, useCameraDevice } from 'react-native-vision-camera'
import { StyleSheet } from 'react-native'

function App() {
  const device = useCameraDevice('back')

  if (device == null) return <NoCameraDeviceError />
  return (
    <Camera
      style={StyleSheet.absoluteFill}
      device={device}
      isActive={true}
    />
  )
}
```
