## Quick example

Mount a camera preview with just a few lines of TypeScript:

```tsx
import { Camera, useCameraDevice, useCameraPermission } from 'react-native-vision-camera'
import { StyleSheet } from 'react-native'

function App() {
  const { hasPermission, requestPermission } = useCameraPermission()
  const device = useCameraDevice('back')

  if (!hasPermission) return <PermissionsPage onPress={requestPermission} />
  if (device == null) return <NoCameraDeviceError />

  return (
    <Camera
      style={StyleSheet.absoluteFill}
      device={device}
      isActive={true}
    />
  )
}
```

## Installation

<Steps>
  <Step title="Install the package">
    <CodeGroup>
      ```bash npm
      npm install react-native-vision-camera
      ```
      ```bash yarn
      yarn add react-native-vision-camera
      ```
      ```bash expo
      npx expo install react-native-vision-camera
      ```
    </CodeGroup>
  </Step>
  <Step title="Install iOS pods">
    ```bash
    cd ios && pod install
    ```
    <Note>Skip this step for Expo managed workflow.</Note>
  </Step>
  <Step title="Add permissions">
    Add camera (and optionally microphone) permissions to your `Info.plist` (iOS) and `AndroidManifest.xml` (Android). See the [Getting Started guide](/guides/getting-started) for the exact XML snippets.
  </Step>
  <Step title="Mount your camera">
    Use `useCameraDevice` and `useCameraPermission` hooks, then render the `<Camera>` component. See the [full guide](/guides/getting-started#use-the-camera-view) for a complete example.
  </Step>
</Steps>
